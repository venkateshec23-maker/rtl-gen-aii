## Data Flow

1. User inputs prompt in UI
2. LLM generates Verilog & Testbench
3. Testbench is validated for real assertions
4. Verilog compiles and simulates
5. OpenLane runs physical design in isolated timestamped dir
6. Metrics parsed and stored in PostgreSQL
7. UI updates with results and History
