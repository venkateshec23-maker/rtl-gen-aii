## WHAT THE PROJECT DOES

```
Natural Language Input
    ↓
"Create an 8-bit adder"
    ↓
Parse Specifications
    ↓
Generate Optimized Prompt
    ↓
Call LLM (Mock/Claude/DeepSeek)
    ↓
Extract Verilog Code
    ↓
Format & Verify Code
    ↓
Output: Production Verilog
```

**Result:**
```verilog
module adder_8bit(
    input [7:0] a, b,
    output [7:0] sum,
    output carry
);
    assign {carry, sum} = a + b;
endmodule
```
